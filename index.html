About the Project.

Zen Framework, as the name says, is a css framework tha uses a extremely simple semantic and provides a readable css code. It does not provides JS yet, but in futures updates it'll.

The Structure

The structure of the stylesheet is separated by types of classes: Globals, Text-Tags, Block-Tags, etc. It provides a very easy uderestanding of the code. It also can be edited, you just have to open the archive style.css in your IDE.
